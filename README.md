# Groblon
A web application and server that allows saving notes in a single place and access them from anywhere I need at anytime




### Why making this ?
The main reason is because I need something useful at least for myself.
And the second reason is to learn new stuff (Web framework and the V programing language for the server)



Current feature plans:
[ ] Simple Text editing
[ ] Media display (Rendering images, videos, audio content)
